AsaanPay Digital Wallet System

Project Overview

AsaanPay Digital Wallet System is a console based application developed in C++ as an Object Oriented Programming Lab Project.

The system simulates the functionality of modern digital wallet platforms by allowing users to create accounts, manage balances, transfer money, view transaction history, and securely manage their wallet credentials.

Main Features

• User Registration
• Secure Login using Four Digit PIN
• Wallet Top Up
• Cash Out
• Money Transfer Between Users
• Transaction History
• PIN Management
• System Information Dashboard

Object Oriented Programming Concepts Used

• Encapsulation
• Abstraction
• Arrays of Objects
• Static Variables
• Function Overloading
• Constructors
• Getters and Setters

Technologies Used

• C++
• Object Oriented Programming
• String Handling
• Arrays
• Input Validation
